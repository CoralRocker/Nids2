Expand description

Trait converting from a type to a byte array so that it can be written to a file on the disk. Prefer big-endian endianess for consistency. from_bytes can return a different type than Self. That type is T. This is used for certain types, such as str and Ref which cannot themselves be created and returned, but which can be saved to bytes.

Example

impl Saveable<str> for str { /*...*/ } // wouldn't work because `from_bytes` cannot return str as it has no definite size.
impl Saveable<String> for str { /*...*/ } // would work because `String` can be returned.
